• Click on "Choose file" button on the screen and select a Map text file from the Map folder of this project.
  • Now click one of the white cells to create a source node and click again on another white cell to create destination code.
  • Click on start to start the animation of creating path
  • If you want slow animation click decrease interval before clicking start or increase interval to speed up the process.
  • If you want to see each step of A* Algorithm press step instead of start.
  • You will need to refresh the page if you want to start-over.
